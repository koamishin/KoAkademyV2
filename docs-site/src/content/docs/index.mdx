## What is KoAkademyV2?

**KoAkademyV2** is an opinionated, production-ready platform that powers the day-to-day operations of a school or training institute. It bundles a hybrid Vue 3 + Inertia.js user app, a Filament v5 admin panel, and four first-party modules — **Admissions**, **Enrollment**, **Classroom**, and **Blog** — into a single, cohesive Laravel 13 codebase.

It eliminates the repetitive setup phase of integrating a modern frontend (Vue 3, Inertia, Tailwind) with a robust backend (Laravel, Fortify, Filament), so you can ship features immediately.

## Tech Stack at a Glance

- **Backend:** PHP 8.5, Laravel 13, Laravel Octane (FrankenPHP)
- **Frontend:** Vue 3, Inertia.js v2, Tailwind CSS v4, Vite 7
- **Admin:** Filament v5 with `filament-shield`, `filament-passkeys`, `filament-activity-log`
- **Auth & Security:** Laravel Fortify (2FA, passkeys), Spatie Permission, Laravel Impersonate
- **Modules:** nwidart/laravel-modules 13
- **Tooling:** Pest 4, PHPStan 2, Rector 2, Pint 1, Wayfinder
